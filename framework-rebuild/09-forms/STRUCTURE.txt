Component Overview
==================

Core Form System:
  FormFactory      → Creates forms and builders
  FormBuilder      → Configures form structure
  Form             → The form instance with data and children
  FormView         → Presentation model for rendering

Type System:
  FormTypeInterface   → Contract for form types
  AbstractType        → Base implementation
  TextType            → Base text input
  EmailType           → Email input (extends TextType)
  PasswordType        → Password input (extends TextType)
  SubmitType          → Submit button
  FormRegistry        → Manages and caches type instances

Options System:
  OptionsResolver     → Validates and resolves options

HTTP Integration:
  Request             → HTTP request abstraction
  ParameterBag        → Key-value parameter storage

Error Handling:
  FormError           → Represents validation errors
