# Forms

Master Symfony Forms for building robust, validated forms with powerful features and flexible theming.

---

## Learning Objectives

After completing this topic, you will be able to:

- Build forms using Symfony's Form component
- Handle form submissions and validation
- Create custom form types and extensions
- Apply form theming with Bootstrap 5 and Tailwind
- Implement CSRF protection
- Use data transformers for data conversion
- Leverage form events for dynamic behavior
- Work with complex form structures (collections, embedded forms)

---

## Prerequisites

- Symfony Architecture basics
- Controllers and routing knowledge
- Twig templating fundamentals
- Data validation concepts
- Understanding of HTTP POST/GET requests
